#######################################

This README file refers to the data reported in the following article

Title: Local parasitoid adaptations are concealed within larger variations of generic symbiont defenses
Authors: Y. Henry, M. Dahirel, J. Wallisch, P. Rodriguez, S. Ginesi, C. Vorburger

The study design is explained in the publication. For questions please contact henry.youn@orange.fr

#######################################

The dataset contains 5 data files

*Aphid_field_samples_2020
*Data_parasitism_2020
*Lfabarum_field_samples_2020
*Microsatellite_genotyping_Lfabarum_2020
*Transfected_aphid_lines_2020

The content of every data file is explained in the top rows of each file.
